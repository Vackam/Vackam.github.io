가능한 모든 조합을 구하라

푸는 방법

정석적인 백트래킹으로 풀면 된다.

다음은 코드이다.

class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        int size = nums.size();
        vector<vector<int>> result;

        auto dfs = [&] (auto& self, int now) -> void {
            // 탈출 조건
            if (now == size) {
                vector<int> v;
                for (auto a : nums) {
                    if (a != 11) v.push_back(a);
                } 

                result.push_back(v);
                return;
            }

            int temp = nums[now];
            self(self, now + 1); // 포함먼저
            nums[now] = 11;
            self(self, now + 1); // 배제 이후에
            nums[now] = temp;

            return;
        }; 

        dfs(dfs, 0);
        return result;
    }
};

배열을 만들고 공유하는 것이 시간이 오래 걸릴 것 같아 사용하지 않는 값을 11로 체크하며 결과가 되었을 때만 배열을 생성하도록 했다.

정답 코드를 보니 더욱 단순하게 만들었다.

다음은 정답 코드이다.

class Solution {
public:
    vector<vector<int>> answer;
    void backtrack(int start, vector<int>& nums, vector<int>& path){
        answer.push_back(path);

        // for all the choices
        for (int i = start; i < nums.size(); i++){
            // Choose 
            path.push_back(nums[i]);
            // Recurse
            backtrack(i + 1, nums, path);

            path.pop_back();
        }
    }
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<int> initPath;
        backtrack(0, nums, initPath);
        return answer;
    }
};

그냥 path를 넘기면서 하니 더욱 코드가 깔끔해졌다.

결론

백트래킹에서 visited를 두어 하는 방법 밖에 몰랐는데, 그냥 넘기는 방식도 배워가는 좋은 문제였다.
