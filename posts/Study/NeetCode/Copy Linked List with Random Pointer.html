Copy Linked List with Random Pointer

2026-06-30 ps

서론

| https://neetcode.io/problems/copy-linked-list-with-random-pointer/question

링크드 리스트인데, 랜덤포인터가 하나 더 있는 것을 deep copy 하시오

푸는 방법

해싱이 중요하다. linear search를 하면서 풀고 싶었기에 해싱을 해서 바로 찾으면 되겠다고 생각하여 각 노드의 val 값을 기준으로 해싱하려 했는데, 문제 조건에 val은 유일하지 않을 수 있다고 하여 실패했다.

그러면 어쩔 수 없이 Node* 그 자체를 해싱해야 한다.

그래서 map을 세우고, 이를 해싱하면서 풀었더니 답이 나왔다.

아래는 코드이다.

    Node* copyRandomList(Node* head) {
        // 람다함수 만들어서 하고 싶은데
        unordered_map<Node* , Node*> m;

        function<Node*(Node*)> recursive_find = [&](Node* target) -> Node* {
            if (!target) return nullptr;
            if (m.find(target) != m.end()) return m[target];
            Node* new_target = new Node(target->val);
            m[target] = new_target;

            new_target->next = recursive_find(target->next);
            new_target->random = recursive_find(target->random);
            
            return new_target;
        };

        return recursive_find(head);

그리고 람다 함수를 recursive 하게 쓰려고 했다가 문제가 생겼다. auto를 사용하여 처음에 작성하였는데, auto가 되어있는 람다 함수를 재귀적으로 사용하면 에러가 났다. 그래서 저렇게 function ... 부분을 작성하였다.

이후에 찾아보니 다음과 같은 식으로도 회피가 가능하다고 한다.

auto recursive_find = [&](auto& self, Node* target) -> Node* {
	if (!target) return nullptr;
	if (m.find(target) != m.end()) return m[target];
	
	Node* new_target = new Node(target->val);
	m[target] = new_target;

	// ★ 호출할 때 자기 자신(self)을 같이 넘겨줍니다.
	new_target->next = self(self, target->next);
	new_target->random = self(self, target->random);
	
	return new_target;
};

이렇게 self로 변수를 직접 받아서 하는 형식이다. 꼬리재귀처럼 계산해야 하는 것을 직접 넣어주는 느낌이다.

이후에 답지를 봤는데, 재밌는 풀이가 있어서 가져왔다.

map과 같이 해싱을 위해 자료형을 쓰게 되면 아무래도 공간 복잡도가 상승하게 된다.

그래서 공간복잡도를 줄이는 방법이, 자신의 복사본을 자신의 다음으로 설정하는 것이다.

예를들어,

\(A->B->C\) 는 복사본 프라임에 대해 \(A->A'->B->B'->C->C'\)을 가지게 된다.

이렇게 처음 세팅을 해 두고 랜덤을 찾을 때,

\(A.random.next == A'.random == A.next.random\) 과 같은 형태가 되어 계산과 공간이 절약되게 된다.

아래는 코드이다.

class Solution {
public:
    Node* copyRandomList(Node* head) {
        if (head == nullptr) {
            return nullptr;
        }

        Node* l1 = head;
        while (l1 != nullptr) {
            Node* l2 = new Node(l1->val);
            l2->next = l1->next;
            l1->next = l2;
            l1 = l2->next;
        }

        Node* newHead = head->next;

        l1 = head;
        while (l1 != nullptr) {
            if (l1->random != nullptr) {
                l1->next->random = l1->random->next;
            }
            l1 = l1->next->next;
        }

        l1 = head;
        while (l1 != nullptr) {
            Node* l2 = l1->next;
            l1->next = l2->next;
            if (l2->next != nullptr) {
                l2->next = l2->next->next;
            }
            l1 = l1->next;
        }

        return newHead;
    }
};
