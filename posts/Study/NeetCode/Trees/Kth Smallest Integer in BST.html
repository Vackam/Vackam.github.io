Kth Smallest Integer in BST

2026-07-07 ps트리dfs

서론

| https://neetcode.io/problems/kth-smallest-integer-in-bst

BST에서 k번쨰 작은 수를 구하라

푸는 방법

BST기 때문에 단순하게 생각해서 inorder 순회를 사용하면서 queue에 값을 저장하고, 나중에 k개 만큼 빼면 그것이 정답이다.

다음은 코드이다.

class Solution {
public:
    int kthSmallest(TreeNode* root, int k) {
        // 중위 순회 하면서 값만 저장
        queue<int> q;

        auto inorder = [&q](auto& self, TreeNode* root) {
            if (root == nullptr) return;

            self(self, root->left);
            q.push(root->val);
            self(self, root->right);
        };

        inorder(inorder, root);
        
        for (int i=0; i<k-1; ++i) q.pop();

        return q.front();
    }
};

다른 방식으로는 dfs를 동일하게 inorder로 순회하면서 카운팅을 하다가 k번째가 되면 중단하고 바로 값을 내는 방식이 있다.

위 코드에 적용하고 싶으면 q.size()를 재귀마다 체크해주면 되겠다.

결론

BST 성질을 이용하면 쉽게 풀 수 있는 문제이다.
