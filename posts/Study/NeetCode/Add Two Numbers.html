Add Two Numbers

2026-06-30 ps링크드리스트

서론

| https://neetcode.io/problems/add-two-numbers/question

거꾸로 정렬된 링크드 리스트를 더해서 해결하라

푸는 방법

컴공 1학년이 배울법한 덧셈기를 만들어서 풀면 된다.

다만 주의할 점이 두 리스트의 길이가 다른 경우가 있다. 필자도 제출해보고 알았다.

그래서 만약 한 쪽이 끝났으면, 계속 값이 없는 상태로 유지할 수 있는 장치가 필요하다.

아래는 코드이다.

ListNode* addTwoNumbers(ListNode* l1, ListNode* l2) {
	ListNode* head = new ListNode();
	int upclimber = 0;

	ListNode* copy_head = head;
	while (l1 || l2) {
		int value = (l1 ? l1->val : 0) +
					(l2 ? l2->val : 0) +
					 upclimber;

		if (value > 9) {
			upclimber = 1;
			value -= 10;
		} 
		else {upclimber = 0;}

		ListNode* new_Node = new ListNode(value);

		head->next = new_Node;
		head = head->next; 

		l1 = l1 ? l1->next : l1;
		l2 = l2 ? l2->next : l2;
	}    

	if (upclimber >= 1) {
		ListNode* new_Node = new ListNode(upclimber);
		head->next = new_Node; 
	}

	return copy_head->next;
}

삼항 연산자를 적극 활용하였다.

결론

굉장히 쉬운 문제였다. 다만, 삼항 연산자가 너무 반복적으로 나와 조금 그랬다. 이럴 때마다 제일 싫어하는 언어인 코틀린이 점점 그리워 진다.
