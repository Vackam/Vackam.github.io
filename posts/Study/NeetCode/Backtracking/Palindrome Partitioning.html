Palindrome Partitioning

2026-07-17 ps백트래킹

서론

| https://neetcode.io/problems/palindrome-partitioning/question

주어진 문자열을 잘랐을 때 나올 수 있는 모든 팰린드롬을 구하라.

푸는 방법

칸막이 방식을 생각하면 된다.

n번째까지 보고 칸막이를 세운 다음 그 다음부터 또 자를 준비를 한다.

이거를 구현하는 방법은 아주 단순하게 칸막이를 세우고 dfs를 호출하면 된다.

다음은 코드이다.

class Solution {
public:
    bool is_palindrome(string s, int start, int end) {
        while (start < end) {
            if (s[start] != s[end]) return false;
            start += 1;
            end -= 1;
        }

        return true;
    }

    void dfs (const string& s, int start, vector<string>& path, vector<vector<string>>& result) {
        if (start == s.size()) {
            result.push_back(path);
            return;
        }

        for (int end = start; end < s.size(); ++end) {
            if (is_palindrome(s, start, end)) {
                path.push_back(s.substr(start, end - start + 1));
                dfs(s, end+1, path, result);
                path.pop_back();
            }
        }
    }
    vector<vector<string>> partition(string s) {
        vector<vector<string>> result;
        vector<string> path;
        dfs(s, 0, path, result);
        return result;
    }
};

주목할 부분은 dfs 파트이다.

시작 지점에서 출발하여 주어진 문자열의 끝까지 탐색해보면서 현재 위치에 칸막이를 세울 수 있는 지 확인한다.

가능한 경우 -> 칸막이를 세우고 문자열을 저장한 다음, dfs를 돌린다.

불가능한 경우 -> 무시하고 다음 가능성을 탐구한다.
