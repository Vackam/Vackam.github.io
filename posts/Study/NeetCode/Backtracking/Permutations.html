주어진 배열의 순열을 구하라.

푸는 방법

정석적으로는 자기 자신과 i번째 칸에 있는 친구를 swap하고 dfs를 진행한 다음 swap을 원상복구 시켜놓은 다음, i += 1을 한다.

즉, 숫자를 계속 바꿔가면서 트리를 만든다고 생각하는 것이다.

하지만, c++에는 무적의 함수 next_permutation이 있다.

다음은 코드이다.

class Solution {

public:

    vector<vector<int>> permute(vector<int>& nums) {
        sort(nums.begin(), nums.end());

        vector<vector<int>> result;

        do {
            result.emplace_back(nums);
        } while (next_permutation(nums.begin(), nums.end()));
        return result;
    }
}

정렬해주고, next_permutation 돌리면 된다.

결론

좋은 무기를 활용하는 것도 능력이다.
