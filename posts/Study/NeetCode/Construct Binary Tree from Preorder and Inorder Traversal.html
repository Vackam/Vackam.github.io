Construct Binary Tree from Preorder and Inorder Traversal

2026-07-07 ps트리

서론

| https://neetcode.io/problems/binary-tree-from-preorder-and-inorder-traversal/

Inorder 순회와 Preorder 순회가 주어졌을 때 트리를 구성하시오.

푸는 방법

백준에도 동일한 문제가 있었던 걸로 기억한다.

이는 각 순회의 특징을 알면 쉽게 풀 수 있다.

Preorder의 경우 "자기자신 -> L -> R" 순으로 간다.

즉, 맨 앞에 있는 값이 root 값이다.

Inorder의 경우 "L -> 자기자신 -> R" 순으로 간다.

Preorder에서 얻은 root 값을 알면 L과 R의 크기를 알 수 있다.

L의 크기를 x라고 한다면,

Preorder에서 R의 위치를 알 수 있고, L의 root와 R의 root는 처음 값이다.

이를 재귀로 풀면 된다.

코드는 다음과 같다.

class Solution {
public:
    TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
        int size = preorder.size();
        auto reconstruct = [&](auto& self, int pre_index, int in_start, int in_end) -> TreeNode* {
            if (in_start > in_end) return nullptr;
            int root_val = preorder[pre_index];
            TreeNode* root = new TreeNode(root_val);

            auto it = find(inorder.begin() + in_start,
                            inorder.begin() + in_end + 1,
                            root_val);
            int index = distance(inorder.begin(), it);

            // 값이 없진 않을 거임.
            int left_size = index - in_start;

            root->left = self(self, pre_index+1, in_start, index-1);
            root->right = self(self, pre_index + 1 + left_size, index + 1, in_end);

            return root;
        };

        return reconstruct(reconstruct, 0, 0, size - 1);
    }
};
