Subtree of Another Tree

2026-07-04 ps트리

서론

| https://neetcode.io/problems/subtree-of-a-binary-tree/question

서브 트리가 있는 지 확인하라.

푸는 방법

단순하게 brute force로 풀어도 된다.

하지만 좀 더 재밌는 방식이 있는데, 문자열로 만들어서 푸는 방법이다.

먼저 root를 전위 순회하여 문자열로 만든다.

이후 sub root를 전위 순회하여 문자열로 만든다.

root의 문자열이 sub root의 문자열을 포함하고 있으면 통과 아니면 실패이다.

다음은 코드이다.

private:
    void serialize(TreeNode* node, string& s) {
        if (node == nullptr) {
            s += "^";
            return;
        }

        s += "#" + to_string(node->val);
        serialize(node->left, s);
        serialize(node->right, s);
    }
public:
    bool isSubtree(TreeNode* root, TreeNode* subRoot) {
        string root_str = ""; 
        string sub_root_str = ""; 

        serialize(root, root_str);
        serialize(subRoot, sub_root_str);

        return root_str.find(sub_root_str) != string::npos;

대신 숫자를 문자열화 하므로 delimeter가 무조건 필요하다. 여기서는 \# 이다.

결론

강력한 stl인 find 등을 활용할 수 있게 문제 풀이를 구성하면 좋을 것이다.
