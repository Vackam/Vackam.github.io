Count Good Nodes in Binary Tree

2026-07-06 ps트리dfs

서론

| https://neetcode.io/problems/count-good-nodes-in-binary-tree/

루트부터 자기 자신까지 자기보다 큰 값이 없는 경우 good node라 하고 주어진 트리에 대해 good node 개수를 세어라

푸는 방법

처음에는 vector를 계속 넘겨서 is_sorted()를 통해 확인하려 했다.

근데 그렇게 하면 값 삽입 삭제가 빈번해서 오래 걸릴 것 같았고, 문제 조건 상 정렬이 안되어있어도 답이 되는 경우가 있기 때문에 바로 폐기 했다.

단순하게 dfs 돌면서 가장 큰 값을 기록해나가면 된다.

아래는 코드이다.

class Solution {
public:
    int goodNodes(TreeNode* root) {
        // 그게 문젠데, 내 path를 어떻게 생각할 지.
        // 그럼 걍 vector<int> 해가지고 전달 전달?
        // dfs로 풀어야 하고

        int result = 0; // root 하나
        auto dfs = [&result](auto& self, TreeNode* root, int biggest_in_path) {
            if (root == nullptr) return;

            if (biggest_in_path <= root->val) {
                result++;
                biggest_in_path = root->val;
            }

            self(self, root->left, biggest_in_path);
            self(self, root->right, biggest_in_path);
        };  

        int biggest_in_path = -101;
        dfs(dfs, root, biggest_in_path);

        return result; 
    }
};
