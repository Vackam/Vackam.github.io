C++GraphicsRenderer

C++ 렌더러 만들기 — 시작

왜 직접 만드나?

DirectX, OpenGL, Vulkan 래퍼를 쓰는 것도 좋지만, 렌더링 파이프라인을 손으로 짜면서 배우는 것만큼 빠른 길은 없다.

목표

  • CPU 소프트웨어 래스터라이저로 시작
  • 삼각형 → 텍스처 → 조명 순으로 확장
  • 이후 Vulkan으로 포팅

래스터라이저 핵심

void rasterize(Triangle& tri, Framebuffer& fb) {
    auto [v0, v1, v2] = tri.verts;
    // bounding box
    int minX = std::min({v0.x, v1.x, v2.x});
    int maxX = std::max({v0.x, v1.x, v2.x});
    // ...
}

Edge function으로 픽셀이 삼각형 안에 있는지 판별하는 게 핵심이다.
