Find Median From Data Stream

중앙값을 찾아라

푸는 방법

가장 간단한 방법은 정렬해서 중앙값을 찾는 것이다.

당연히 각 시도마다 O(nlogn)의 시간 복잡도를 가진다.

여기서 원하는 풀이는 아마도 두 개의 힙을 세우는 것이다

low heap과 max heap을 세워서 편하게 찾는 것이다.

이는 코드를 보면 명확하니 다음 코드를 보자.

class MedianFinder {
public:
    MedianFinder() {
        
    }
    
    void addNum(int num) {
        lo.push(num);
        hi.push(lo.top());
        lo.pop();

        if (hi.size() > lo.size()) {
            lo.push(hi.top());
            hi.pop();
        }        
    }
    
    double findMedian() {
        if (lo.size() > hi.size())
            return lo.top();
    
        return (lo.top() + hi.top()) / 2.0;
    }
private:
    priority_queue<int> lo;
    priority_queue<int, vector<int>, greater<int>> hi;
};

절반의 작은 쪽은 lo에 큰 쪽은 hi에 넣는다.

그리고 lo >= hi를 항상 유지한다. (1개 차이)

결론

첫 풀이 때 두 개 힙을 생각은 했는데, 이걸 밸런스 있게 넣는 방법을 생각해내지 못해서 포기했었다. 이번 풀이를 통해 밸런스 있게 만드는 방법을 배워 좋다.
