주어진 트리가 BST인 지 확인하라

푸는 방법

BST는 왼쪽 오른쪽이 자신의 부모에 따라 작고 큼이 결정되는 트리이다.

그래서 upper_bound와 lower_bound를 잘 트래킹하면 무리 없이 풀 수 있다.

다음은 코드이다.

class Solution {
public:
    bool isValidBST(TreeNode* root) {
        auto check_validation = [](auto& self, TreeNode* root, int lower_bound, int upper_bound) {
            if (root == nullptr) return true;

            int value = root->val;
            if ( !(value > lower_bound && value < upper_bound) ) return false;

            return self(self, root->left, lower_bound, value) && self(self, root->right, value, upper_bound);
        };

        return check_validation(check_validation, root, -1001, 1001);
    }
};

간단하게 재귀함수를 작성하여 풀었다.
