Reverse Nodes in K-Group

2026-07-02 ps링크드리스트

서론

| https://neetcode.io/problems/reverse-nodes-in-k-group/question?list=neetcode150

k 단위로 링크드 리스트를 뒤집자.

푸는 방법

이 문제를 풀면서 아주 상세하게 느낀 점은 포인터를 많이 두지 말자는 것이다.

3개 이상 되면 반복문 안에서 얽힐 수 밖에 없다. 3개 이상일 시 뭔가 잘못되었음을 느끼고 다시 코드를 읽자.

푸는 방식은 재귀로 단순하게 풀면 된다.

다음은 코드이다

class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode* curr = head;
        for (int i = 0; i < k; ++i) {
            if (!curr) return head; // k개가 안 되면 뒤집지 않고 원본 반환 (재귀 종료)
            curr = curr->next;
        }

		// k 만 바꾸기
        ListNode* prev = nullptr;
        curr = head;
        for (int i = 0; i < k; ++i) {
            ListNode* next_node = curr->next;
            curr->next = prev;
            prev = curr;
            curr = next_node;
        }
		
		// 재귀로 연결
        head->next = reverseKGroup(curr, k);

        return prev;
    }
};

결론

많이 헤맸던 부분이 한 함수 안에서 무조건 k 개를 다 해결하려고 했던 것이다. 재귀를 적극 활용하여 있는 k개만 뒤집고, 나머지는 뒤에 애들한테 맞기는 방식을 적극 활용하자.
