Remove Nth Node From End of List

— ps

서론

뒤에서 n 번째에 있는 숫자를 지운 링크드 리스트를 제시하시오

푸는 방법

제일 처음에 푼 방식은 단순히 length를 구한 다음 푸는 방식이다.

탐색은 \(O(n)\) 이고, 제거하는 곳까지 가는 것도 \(O(n)\) 이므로 상관없다고 판단.

아래는 코드이다.

class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        int length = 0;
        ListNode* copy_head = head;
        while (copy_head) {
            length++;
            copy_head = copy_head->next;
        }    

        if (length <= 1) {
            return nullptr;
        }
        else if (length == n) {
            head = head->next;
            return head;
        }
        copy_head = head;
        for (int i=0; i<length-n-1; ++i) {
            copy_head = copy_head->next;
        }

        copy_head->next = copy_head->next->next;

        return head;
    }
};

다른 풀이 방식은 투 포인터 방식이 있다.

slow 변수와 fast 변수를 두고, 이를 dummy를 가리키도록 한다.

dummy는 문제에서 주어진 head를 next로 가지는 변수이다.

(dummy.next = head)

그리고 slow와 fast의 거리 차이를 n만큼 벌려준 상태로, fast가 끝에 도달할 때까지 fast와 slow를 한 칸씩 이동시킨다.

끝에 도달하게 되면, slow의 next가 지워야 할 변수가 된다.

아래는 코드이다.

class Solution {
public:
    ListNode* removeNthFromEnd(ListNode* head, int n) {
        ListNode dummy(0, head);

        ListNode* s = &dummy;
        ListNode* f = &dummy;

        for (int i=0; i<n+1; ++i) f = f->next;
        while (f) {
            s = s->next;
            f = f->next;
        }

        s->next = s->next->next;

        return dummy.next;
    }
};
