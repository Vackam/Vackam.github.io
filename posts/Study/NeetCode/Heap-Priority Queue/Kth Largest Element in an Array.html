Kth Largest Element in an Array

2026-07-21 ps힙우선순위큐

서론

| https://neetcode.io/problems/kth-largest-element-in-an-array/question

정렬되어 있지 않은 배열의 k 번째 큰 값을 찾아라.

푸는 방법

당연히 sort를 하면 풀린다.

다음은 코드이다

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        sort(nums.rbegin(), nums.rend());
        return nums[k-1];    
    }
};

물론 이렇게 하라고 만든 것이 아니다.

우선순위큐를 만들어서 k가 넘어갈 시 계속 pop 해주면서 트래킹 해주면 된다.

다음은 코드이다.

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        priority_queue<int, vector<int>, greater<int>> pq;

        for (auto& a : nums) {
            pq.push(a);

            if (pq.size() > k) pq.pop();
        } 

        return pq.top();
    }
};
