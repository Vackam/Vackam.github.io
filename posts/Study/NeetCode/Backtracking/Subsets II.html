푸는 방법

다른 백트래킹과 동일한데, result에 넣는 조건이 없다는 것이 차이점이다.

for문으로 돌면서 dfs를 진행하는데, 될 때마다 result에 푸쉬해준다.

다음은 코드이다.

class Solution {
    int size;
    vector<vector<int>> result;
public:
    void dfs (int index, vector<int>& path, vector<int>& nums) {
        result.push_back(path);

        for (int i=index; i<size; ++i) {
            if (i > index && nums[i] == nums[i-1]) continue;
            path.push_back(nums[i]);
            dfs(i + 1, path, nums);
            path.pop_back();
        }
        return;
    }
    vector<vector<int>> subsetsWithDup(vector<int>& nums) {
        size = nums.size();
        vector<int> path;
        sort(nums.begin(), nums.end());
        dfs(0, path, nums);
        return result;
    }
}

사실 종료 조건을 찾는 게 너무 힘들었다. 공책에 적어가면서 해야겠다.
