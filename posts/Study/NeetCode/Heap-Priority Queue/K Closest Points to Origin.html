K Closest Points to Origin

2026-07-20 ps우선순위큐

서론

| https://neetcode.io/problems/k-closest-points-to-origin/question?list=neetcode150

원점이랑 가까운 k개의 점들을 반환하라

푸는 방법

처음에 k번째인 데 겹치는 동일한 값에 대해서는 따로 처리해야 하는 줄 알아서 온갖 복잡한 짓을 다해봤다.

근데 그게 아니라, k개를 전부 반환하기만 하면 되더라.

그러면 간단하게 우선순위 큐를 세우고 조지면 된다.

다음은 코드이다.

using pivi = pair<int, vector<int>>;

class Solution {
public:
    int dis_square(const vector<int>& v) {
        return (v[0]*v[0] + v[1]*v[1]);
    }
    vector<vector<int>> kClosest(vector<vector<int>>& points, int k) {
        priority_queue<pivi> pq;
        vector<vector<int>> result;

        for (auto& a : points) {
            int dis = dis_square(a);
            pq.push({-dis, a});
        }

        for (int i=0; i<k; ++i) {
            auto a = pq.top();
            pq.pop();
            result.push_back(a.second);
        }

        return result;
    }
};

거리를 구하는 데 굳이 시간이 많이 드는 연산인 sqrt를 할 필요가 없다.

또한, 기본 priority_queue는 max heap 이므로, distance를 넣어줄 때 음수로 넣어주어 min heap 처럼 작동하게 하였다.

결론

역시 문제를 푸는 것보다 독해가 훨씬 중요하다.
