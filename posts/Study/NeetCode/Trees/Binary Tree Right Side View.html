Binary Tree Right Side View

2026-07-06 ps트리bfs

서론

| https://neetcode.io/problems/binary-tree-right-side-view/

트리를 오른편에서 봤을 때 vector 값을 찾아라.

푸는 방법

Binary Tree Level Order Traversal

위 문제 풀이를 조금 바꾸면 해결이 된다.

맨 오른쪽에 있다는 것을 생각해보면, queue에 left -> right 순으로 적재했을 때 queue 마지막 값이 해당 depth에 가장 오른쪽 값이 된다.

그래서 나머지 값은 확인하지 않고 마지막 값만 찾으면 된다.

다음은 코드이다.

class Solution {
public:
    vector<int> rightSideView(TreeNode* root) {
        if (root == nullptr) return {};
        vector<int> result;
        queue<TreeNode*> q;

        q.push(root);

        while (!q.empty()) {
            int size = q.size();
            for (int i=0; i<size - 1; ++i) {
                auto temp = q.front();
                q.pop();

                if (temp->left != nullptr) q.push(temp->left);
                if (temp->right != nullptr) q.push(temp->right);
            }
            // 마지막 한 칸 전까지만 진행하고,
            // 마지막 녀석은 무조건 오른쪽에 있는 놈이기 때문에
            // 얘만 저장
            auto temp = q.front();
            q.pop();

            result.push_back(temp->val);
            if (temp->left != nullptr) q.push(temp->left);
            if (temp->right != nullptr) q.push(temp->right);
        } 

        return result;
    }
};
