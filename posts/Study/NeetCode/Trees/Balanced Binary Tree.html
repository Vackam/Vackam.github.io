Balanced Binary Tree

2026-07-04 ps트리

서론

| https://neetcode.io/problems/balanced-binary-tree/question

트리가 균형되게 되어있는 지 확인하라.

푸는 방법

백준 같은 곳에서는 리스트를 주면서 했을 거 같은데, 트리가 들어와 버리니까 조금 당황했다.

푸는 방법은 단순하다. 재귀함수로 자식들의 균형이 이루어져 있는 지 확인해보면 된다.

코드는 다음과 같다.

bool isBalanced(TreeNode* root) {
	auto check_height = [](auto& self, TreeNode* node) -> int {
		if (!node) return 0;

		int left_height = self(self, node->left);
		if (left_height == -1) return -1;

		int right_height = self(self, node->right);
		if (right_height == -1) return -1;

		if (abs(left_height - right_height) > 1) return -1;

		return 1 + max(left_height, right_height);
	};

	return check_height(check_height, root) != -1;
}

왼편과 오른편의 높이 차이가 1보다 크게 된다면, 균형이 안맞다는 뜻이므로 -1을 반환하도록 한다.

결론

기본 자료형에 익숙해질 필요가 있다.
